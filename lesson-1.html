Урок 1: Введение в JavaScript

15 минут Начинающий

Что такое JavaScript?

JavaScript — это язык программирования, который используется для создания интерактивных веб-страниц. Вместе с HTML и CSS он формирует основу современного веб-разработки.

JavaScript позволяет:

  • Добавлять интерактивность на сайт (анимации, всплывающие окна)
  • Обрабатывать действия пользователя (клики, ввод текста, движение мыши)
  • Изменять содержимое страницы без перезагрузки
  • Отправлять запросы на сервер и получать данные
  • Создавать сложные веб-приложения и игры

Подключение JavaScript к странице

Существует два способа подключить JavaScript к HTML-странице:

  • Внутренний (inline) - код пишется внутри тега <script>
  • Внешний (external) - код выносится в отдельный файл с расширением .js

Пример внутреннего подключения:


<script>
    console.log("Привет, мир!");
</script>

Пример внешнего подключения:


<script src="script.js"></script>

Тег <script> рекомендуется размещать перед закрывающим тегом </body>, чтобы скрипты не блокировали загрузку страницы.

Переменные

Переменные используются для хранения данных. В JavaScript есть три способа объявления переменных:

  • let — для переменных, значение которых может меняться
  • const — для переменных, значение которых не меняется (константы)
  • var — устаревший способ (использовать не рекомендуется)

Примеры объявления переменных:


// Объявление переменной с помощью let
dlet userName = "Иван";
let age = 25;
let isStudent = true;

// Объявление константы
const PI = 3.14159;
const SITE_NAME = "JavaScript Pro";

// Изменение значения переменной
userName = "Петр";
age = age + 1;

console.log(userName); // Выведет: Петр
console.log(age);      // Выведет: 26
console.log(PI);       // Выведет: 3.14159

Типы данных

В JavaScript есть 8 основных типов данных:

  • number — числа (целые и дробные)
  • string — строки (текст в кавычках)
  • boolean — логический тип (true или false)
  • undefined — значение не определено
  • null — пустое значение
  • object — объекты (сложные структуры)
  • array — массивы (списки значений)
  • symbol — уникальные идентификаторы

Примеры разных типов данных:


let number = 42;                    // number
let text = "Привет, мир!";          // string
let isActive = true;                // boolean
let nothing = null;                 // null
let notDefined;                     // undefined

let user = {                        // object
    name: "Анна",
    age: 30,
    city: "Москва"
};

let colors = ["red", "green", "blue"]; // array

console.log(typeof number);    // "number"
console.log(typeof text);      // "string"
console.log(typeof isActive);  // "boolean"
console.log(typeof user);      // "object"
console.log(typeof colors);    // "object" (массив - это объект)

Операторы

Операторы выполняют действия с данными:

Арифметические операторы:


let a = 10;
let b = 3;

console.log(a + b);  // 13 (сложение)
console.log(a - b);  // 7  (вычитание)
console.log(a * b);  // 30 (умножение)
console.log(a / b);  // 3.333... (деление)
console.log(a % b);  // 1  (остаток от деления)
console.log(a ** b); // 1000 (возведение в степень)

Операторы сравнения:


console.log(5 > 3);   // true
console.log(5 < 3);   // false
console.log(5 >= 5);  // true
console.log(5 <= 3);  // false
console.log(5 == "5"); // true (сравнение с приведением типов)
console.log(5 === "5"); // false (строгое сравнение)
console.log(5 != "5");  // false
console.log(5 !== "5"); // true

Логические операторы:


let x = true;
let y = false;

console.log(x && y); // false (И)
console.log(x || y); // true  (ИЛИ)
console.log(!x);     // false (НЕ)
