Урок 2: Условные операторы и циклы

20 минут Начинающий

Условные операторы

Условные операторы позволяют выполнять разные блоки кода в зависимости от условий. Это основа принятия решений в программировании.

Оператор if

Выполняет код, только если условие истинно (true).


let age = 18;

if (age >= 18) {
    console.log("Вы совершеннолетний");
    console.log("Вам доступен весь контент");
}

Оператор if...else

Выполняет один блок кода, если условие истинно, и другой — если ложно.


let age = 16;

if (age >= 18) {
    console.log("Вы совершеннолетний");
} else {
    console.log("Вы несовершеннолетний");
    console.log("Доступ ограничен");
}

Оператор else if

Позволяет проверять несколько условий последовательно.


let score = 85;

if (score >= 90) {
    console.log("Отлично! Ваша оценка: 5");
} else if (score >= 70) {
    console.log("Хорошо! Ваша оценка: 4");
} else if (score >= 50) {
    console.log("Удовлетворительно! Ваша оценка: 3");
} else {
    console.log("Нужно больше практики. Оценка: 2");
}

Тернарный оператор

Краткая форма записи if...else для простых условий.


let age = 20;
let status = age >= 18 ? "Совершеннолетний" : "Несовершеннолетний";
console.log(status); // "Совершеннолетний"

// То же самое с if...else:
if (age >= 18) {
    status = "Совершеннолетний";
} else {
    status = "Несовершеннолетний";
}

Оператор switch

Удобен для проверки одного значения с множеством вариантов.


let day = "понедельник";

switch(day) {
    case "понедельник":
        console.log("Начало рабочей недели");
        break;
    case "пятница":
        console.log("Почти выходные");
        break;
    case "суббота":
    case "воскресенье":
        console.log("Выходной! Отдыхаем");
        break;
    default:
        console.log("Обычный рабочий день");
}

Циклы

Циклы позволяют выполнять один и тот же код многократно. Это избавляет от дублирования кода.

Цикл for

Самый часто используемый цикл. Состоит из трех частей: инициализация, условие, инкремент.


// Вывод чисел от 1 до 5
for (let i = 1; i <= 5; i++) {
    console.log("Число:", i);
}

// Перебор массива
let fruits = ["яблоко", "банан", "апельсин", "манго"];
for (let i = 0; i < fruits.length; i++) {
    console.log("Фрукт:", fruits[i]);
}

// Сумма чисел от 1 до 10
let sum = 0;
for (let i = 1; i <= 10; i++) {
    sum = sum + i;
}
console.log("Сумма от 1 до 10:", sum); // 55

Цикл while

Выполняется, пока условие истинно. Условие проверяется перед каждой итерацией.


let count = 1;
while (count <= 5) {
    console.log("Итерация:", count);
    count++;
}

// Поиск числа
let target = 42;
let guess = 0;
let attempts = 0;

while (guess !== target) {
    guess = Math.floor(Math.random() * 100) + 1;
    attempts++;
}
console.log(`Число ${target} найдено за ${attempts} попыток`);

Цикл do...while

Похож на while, но условие проверяется после выполнения кода. Тело цикла выполнится хотя бы один раз.


let i = 10;
do {
    console.log("Значение:", i);
    i++;
} while (i <= 5);
// Выведет: Значение: 10 (один раз, даже если условие ложно)

Цикл for...of (для массивов)

Удобный способ перебора элементов массива.


let colors = ["красный", "зеленый", "синий", "желтый"];
for (let color of colors) {
    console.log("Цвет:", color);
}

Управление циклами


// break - прерывает цикл
for (let i = 1; i <= 10; i++) {
    if (i === 5) {
        break; // остановка на 5
    }
    console.log(i); // 1, 2, 3, 4
}

// continue - пропускает текущую итерацию
for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        continue; // пропускаем 3
    }
    console.log(i); // 1, 2, 4, 5
}

Практическое задание

1. Напишите программу, которая определяет время суток по часу (0-23):

  • 6-11: "Утро"
  • 12-17: "День"
  • 18-23: "Вечер"
  • 0-5: "Ночь"

2. Используя цикл for, выведите все четные числа от 2 до 20.

3. Используя цикл while, найдите сумму всех чисел от 1 до 100 и выведите результат.
